# Saber Alter RVC Voice Model (v2)

## Introduction / 简介
This repository contains the RVC (Retrieval-based Voice Conversion) model for **Saber Alter** from the *Fate* series.
本仓库包含了《Fate》系列中 **Saber Alter（黑呆）** 的 RVC 语音模型。

## Files / 文件说明
- `Saberalter.pth`: Model weights (RVC v2).
- `added_IVF314_Flat_nprobe_1_Saberalter_v2.index`: Feature index file.

## Usage / 使用方法
1. Download both files from this repository.
2. Load them into your RVC inference software.
3. Perfect for AI cover songs or voice acting projects.
